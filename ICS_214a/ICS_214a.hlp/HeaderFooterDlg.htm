Header/Footer Dialog

Dialog Box

This dialog box attempts to mimic the header information block on a paper ICS-214 form. Fill it in as best that you can. The only interesting fields are the Incident Name (we have multiple instances) and the Operational period.

The Footer Information block here does not include the page number on the ICS-214 because that is generated when the data is output.

When a new log file is created (e.g. when starting the program for the first time or giving a file name that does not exist when starting) the dialog box is started. At any time the information in this dialog box may be reviewed and modified. When the OK button is selected all the data in the program is written to the "214" file.
